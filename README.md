# Venmo engineering portal

## Running it locally

First install Jekyll if you don't already have it.

```sh
$ gem install jekyll
```

Then run

```sh
$ jekyll serve
```

## Writing posts

Posts are written in markdown.  Be sure you include this at the top of each of your posts:

```
---
layout: post
title: "Your Post Title"
author: Your Name
---
```
